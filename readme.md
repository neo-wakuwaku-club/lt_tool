
# Lt-tool
わくわくLTで横にコメント流れたら面白いかなと思って作ってみました。

# 使い方
- タイマー管理画面: http://localhost:3000/control
- 表示画面: http://localhost:3000/display
- 参加者コメントページ: http://localhost:3000/participant

# 準備に必要な環境
- 全画面表示可能なブラウザ（firefox etc,,,,）
- OBS
- node.js

# setup
```
node server.js
```
ローカルサーバー立ち上げたら、vscodeのトンネルに繋げる。

そのURLをみんなに共有しよう
